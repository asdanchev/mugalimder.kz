# Mugalimder.kz

**Mugalimder.kz** — онлайн-платформа для учителей Казахстана.  
Цель проекта — создать пространство для обмена опытом, публикации методических материалов  
и общения между педагогами.

---

##  Технологии

- Python 3.12 / Django 5.1  
- PostgreSQL 16  
- Nginx + Gunicorn + Docker  
- Ubuntu Server (production setup)
